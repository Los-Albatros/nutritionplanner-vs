# NutritionPlanner — Vintage Story Mod

A lightweight **client-only** HUD that tracks your four nutrition bars in real time,
warns you before they drop too low, and suggests what to eat from your inventory.
No server install required — works on any vanilla or modded server.

**By [Los Albatros](https://github.com/Los-Albatros) — MIT License**

---

## Features

- **Live nutrition HUD.** Four colored bars (Grain, Veg, Protein, Dairy) update every
  second directly from the game's watched attributes — no server mod needed.
- **Pulse warning.** Bars flash when a category drops below the warning threshold
  (default 30 %).
- **Chat alert + suggestion.** When a bar falls below the critical threshold (default
  15 %), a chat message appears and the mod suggests an item from your inventory that
  covers the missing category.
- **Meal history.** The HUD tracks every positive nutrition delta (eating) and shows
  a summary of recent meals.
- **Manual suggestion button.** Click **Suggest** inside the HUD, or type
  `/nutrition suggest`, to get an instant recommendation at any time.
- **In-game settings (ConfigLib).** If [ConfigLib](https://mods.vintagestory.at/configlib)
  is installed, a settings panel lets you tune the warning threshold, critical threshold,
  and alert cooldown, and shows your current toggle keybind.
- **Optional AI suggestions (chatAIMod).** If [chatAIMod](https://github.com/roquess/chataimod-vs)
  is installed and configured with an API key, suggestions are generated by the AI instead
  of the local algorithm. Falls back to local if the AI is unavailable or returns an
  incoherent response.
- **Client-only.** `side: Client`, `requiredOnServer: false`. Works on any server,
  including vanilla. Nothing is installed on the server.

## Controls

| Action | Default |
|---|---|
| Toggle HUD | `N` (rebind in **Options → Controls**) |
| `/nutrition suggest` | Request a suggestion from chat |

## Installation

Download `nutritionplanner_v0.1.0.zip` and drop it in your Vintage Story `Mods/` folder
(`%AppData%\VintagestoryData\Mods\` on Windows).

- **Server: not required.** The mod is purely client-side and works on any server.
- **Optional: install on the server** with `requiredOnClient: true` (already set) so VS
  notifies connecting players that the mod is available and prompts them to download it
  (requires the mod to be published on [ModDB](https://mods.vintagestory.at)).

## Configuration

Settings are saved per-player in `ModConfig/nutritionplanner-client.json`.

| Option | Default | Description |
|---|---|---|
| `HudVisible` | `true` | Whether the HUD is shown on startup. |
| `Threshold1` | `30` | Warning threshold % — bars pulse below this value. |
| `Threshold2` | `15` | Critical threshold % — triggers chat alert + suggestion. |
| `ChatCooldownSeconds` | `300` | Minimum seconds between alerts for the same category. |

All options are editable live via **ConfigLib** if installed.

## Optional dependencies

| Mod | Effect |
|---|---|
| [ConfigLib](https://mods.vintagestory.at/configlib) | In-game settings panel |
| [chatAIMod](https://github.com/roquess/chataimod-vs) | AI-powered suggestions instead of local algorithm |

Neither is required. The mod works standalone.

## Building from source

```
dotnet build nutritionPlannerVintageStoryMod.csproj -c Release
dotnet test  ../nutritionPlannerVintageStoryMod.Tests/nutritionPlannerVintageStoryMod.Tests.csproj
```

Requires `VINTAGE_STORY` environment variable pointing at your game install, or adjust
the `<VSPath>` in the `.csproj`.
